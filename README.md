# Organização Financeira Familiar

Sistema web para organização financeira familiar, desenvolvido com Next.js 14, TypeScript, Prisma, NextAuth, Zod, React Hook Form, shadcn/ui, Lucide Icons, ESLint, Prettier, PWA (next-pwa), IndexedDB (Dexie) e @tanstack/react-query.

## Setup Inicial

1. Instale as dependências:
   ```sh
   npm install
   ```
2. Configure as variáveis de ambiente conforme necessário.
3. Para rodar o projeto:
   ```sh
   npm run dev
   ```

## Stack
- Next.js 14 (App Router)
- TypeScript
- Prisma + PostgreSQL (Neon)
- NextAuth (Credentials)
- Zod + React Hook Form
- shadcn/ui + Lucide Icons
- ESLint + Prettier
- PWA (next-pwa)
- IndexedDB (Dexie)
- @tanstack/react-query
